# Binance Futures Order Bot (USDT-M Testnet)

## Overview
This project is a CLI-based trading bot built for Binance USDT-M Futures Testnet using Python.
The bot supports core order types such as Market and Limit orders, along with advanced order
strategies like Stop-Limit and TWAP. It includes proper input validation, structured logging,
and modular code design for reusability.

---

## Features
- Market Orders
- Limit Orders
- Stop-Limit Orders (Advanced)
- TWAP (Time-Weighted Average Price) Strategy (Advanced)
- Input validation for side, quantity, and price
- Structured logging with timestamps
- Binance Futures Testnet support (USDT-M)

---

## Tech Stack
- Python 3.8+
- python-binance (official Binance API SDK)
